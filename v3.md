## I. BỐI CẢNH VÀ LÝ DO CHUYỂN TRẬN
### 1.1 Hiện Trạng V2 (Classification 5 Classes)
| Metric | Giá trị |
|--------|---------|

| V2 Full Macro F1 | **46.79%** (±0.63%) |

| **Master** F1 | 0.67 ✅ |
| **Advanced** F1 | **0.30** 🔴 (Recall = 0.28) |
### 1.2 Tại Sao Classification Là Bẫy?
- **Ranh giới giả tạo:** ELO 1399 vs 1401 — cách nhau 2 điểm nhưng lọt 2 class khác nhau → Model bị phạt 100%.
- **Phong độ dao động:** Một kỳ thủ ELO 1500 có thể đánh ván hôm nay như 1300, ngày mai như 1700. Classification coi mọi lần lệch class là "Sai hoàn toàn" — quá tàn nhẫn.
- **"Cái bụng" nát:** 3 class giữa (Intermediate/Advanced/Expert) chồng chéo cực độ về mặt features. Model không thể tạo hàng rào phân cắt rõ ràng.
### 1.3 Regression Giải Quyết Được Gì?
- **Dự đoán liên tục:** Model trả ra con số ELO cụ thể (vd: 1542) thay vì tên Class.
- **Sai số được cân đo hợp lý:** Dự đoán 1542 cho kỳ thủ ELO 1600 (lệch 58) → vẫn tốt. Classification sẽ phạt nếu chệch Class.
- **Phù hợp bản chất bài toán:** ELO là thang đo liên tục (0 → 3000+). Đo năng lực tức thời = Đoán điểm số, không phải dán nhãn.
- **Metric trực quan:** MAE (Mean Absolute Error) — "Trung bình mỗi ván, model đoán lệch bao nhiêu ELO?" Ai cũng hiểu.
---
## II. MỤC TIÊU V3
| Metric | Mục tiêu | Ghi chú |
|--------|----------|---------|
| **MAE** (Mean Absolute Error) | **≤ 220 ELO** | Dự phóng optimistic ≈ 180, pessimistic ≈ 250 |
| **RMSE** (Root Mean Squared Error) | ≤ 280 ELO | Phạt nặng hơn ván dự đoán lệch cực lớn |
| **R² Score** | ≥ 0.55 | Tỷ lệ phương sai được giải thích (> 0.5 = mô hình có ý nghĩa) |
| **Tương đương Classification** | Accuracy ≥ 55% | Đổi ngược ELO dự đoán → Class rồi so sánh để đối chiếu với V2 |
> **Ngưỡng Thành Công:** MAE ≤ 220 là THẮNG. Nếu ≤ 180, là ĐẠI THẮNG.

## VII. DỰ PHÓNG KẾT QUẢ
### Kịch bản Optimistic (Đại thắng 🟢)
- MAE ≤ 180 ELO
- R² ≥ 0.65
- Regression→Class Accuracy ≥ 55%
- Ý nghĩa: Model nắm bắt rất tốt "phong cách đánh cờ = trình độ"
### Kịch bản Realistic (Chấp nhận được 🟡) 
- MAE ≈ 200-220 ELO
- R² ≈ 0.55-0.60
- Regression→Class Accuracy ≈ 50-53%
- Ý nghĩa: Model hoạt động, phân biệt rõ ràng Beginner vs Master, mờ nhạt ở giữa
### Kịch bản Pessimistic (Cần tiếp tục đầu tư 🔴)
- MAE > 250 ELO
- R² < 0.50
- Ý nghĩa: 11 features Stockfish chưa đủ thông tin, cần bổ sung Sequence Features (TF-IDF Moves) ở V4
